# Notes-App
A full stack Notes application built using the MERN stack that allows users to create, edit, delete, and manage notes efficiently. The application provides a clean and responsive interface for organizing important information with secure backend storage and REST API integration.

---

## ✨ Features

* 📝 Create new notes  
* ✏️ Edit existing notes  
* ❌ Delete notes  
* 📋 View all saved notes  
* 🎯 Clean and responsive UI 

---

## 🛠️ Tech Stack

### Frontend
* React
* JavaScript
* CSS

### Backend
* Node.js
* Express.js
* MongoDB

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/Bhaktishree-Pakka/Notes-App.git
```

2. Navigate to the project folder:

```
cd Notes-App
```

3. Install dependencies:

```
npm install
```
4.Install frontend dependencies:

```
cd notes-frontend
npm install
```

5.Install backend dependencies:

```
cd ../notes-backend
npm install
```

6.Create .env file inside backend folder:

```
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
```

7. Run the development server:

```
npm run dev
```
